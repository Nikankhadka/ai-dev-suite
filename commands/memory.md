# Memory Command

Manage instinct-based learning and skill management: $ARGUMENTS

## Operations

### View Instincts
- `instinct-status` - View learned instincts and their stats
- `projects` - List known projects and instinct stats

### Import/Export
- `instinct-import` - Import instincts from file
- `instinct-export` - Export instincts to file

### Evolve & Promote
- `evolve` - Cluster instincts into reusable skills
- `promote` - Promote project instincts to global scope

### Skills
- `skill-create` - Generate skills from git history patterns
- `learn` - Extract patterns and learnings from session

## Report Format

List current state, changes made, and any pending actions.

**TIP**: Run `/memory instinct-status` to see what patterns have been learned.
