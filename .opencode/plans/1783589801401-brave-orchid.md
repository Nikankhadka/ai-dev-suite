# ECC Agent Orchestration Simplification Plan

## Goal
Reduce from 14 agents → 7 agents, merge commands per plan→build→review→test→document workflow, keep non-engineering tools consolidated under an `ops` agent.

## New Workflow
**plan → build → review → test → document**
