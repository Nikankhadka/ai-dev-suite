# Strip to Vanilla OpenCode Config

## Goal
Remove everything that's custom ECC framework — keep only a minimal opencode config with native integrations.

## What Stays
- `opencode.jsonc` — minimal: build agent, permission, LSP, MCP servers (native integrations)
- `.opencode/` — opencode's own runtime data
- `.gitignore` — from git

## What Goes (not original opencode / not a native integration)
- `instructions/INSTRUCTIONS.md` — ECC orchestration framework (custom)
- `prompts/agents/` — all custom subagent prompts (6 files)
- `commands/` — all custom command templates (10 files)
- `plugins/` — ECC plugin system (3 files + lib/)
- `tools/` — custom tool implementations (8 files)
- `skills/` — custom skill directories (4 dirs)
- `index.ts` — plugin entry point
- `package.json`, `package-lock.json`, `tsconfig.json` — build tooling
- `DEVELOPMENT.md`, `README.md` — project docs
- `node_modules/` — external dependencies

## Verification
1. `opencode.jsonc` parses as valid JSON
2. No dangling file references in config
3. Git status shows clean removal
